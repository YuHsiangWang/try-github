# 本書簡介

台灣電力公司響應[`國發會數位服務個人化(My Data)`](/mydata.md)政策，推出[`我的電力數位服務平台`](https://mydata.taipower.com.tw/)。本書指導軟體開發人員如何透過台電MyData API，存取個人用電資料，提供豐富的用電服務。
